COURSES

Web Development

HTML, CSS, JavaScript (jQuery)

Create personalized websites and web programs in a challenging but supportive environment! Web Development at Vikings Dev will teach you everything you need to know to code in HTML, CSS, and JavaScript. Throughout the program, we will focus on providing numerous opportunities for you to develop and show off your web development skills through open-ended projects.
Starter Kit Resources

Game Design

GameMaker Studio

Design, code, and create your own game with GameMaker: Studio. Learn about good game design, art styles, coding and basic logic, while having a place to learn and see things instantaneously. Feel free to take your game to the limits of your imagination. If you can think of it, we can help you make it.
Starter Kit

Unreal Engine

Unreal Engine 4

Unreal Engine is a Game Engine created by Epic Games and utilized by many professional studios. Unreal Engine has alot of readily available resources, prepared to help you create a professional and detailed end product. We will teach you the basics, provide you with many resources and provide any assistence needed to help you reach your end goal. For the course, you will need to bring your own laptop and we are excited to see what you can create!!
Starter Kit

Discord Bots

Discord Bots

Learn how to make a chat bot in Discord! Learning how to make a Discord Bot is great for learning other skills! In this project, you'll learn how to use Bash, an essential tool in software programming, use APIs, create a database using JSON files, and more. At the end of the year you will have a tangible product, which is a bot that can do things like check the weather, play music, and other cool things. Click the 'Starter Kit' button to find out more!
