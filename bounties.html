STEP 2

Read the Contribution Guidelines

Read up on their README or CONTRIBUTING to see how you can help out their project. For Dentaku, read up on how you can set up a development environment and how to start making commands.

STEP 3

Get Creative or Get Hunting.

Do you have an idea for how to improve the project? Great! Submit an issue, fork the project and start working on your game-changing idea.

You don't have an idea? Start sharpening your knives. You're going bug hunting. Browse the Issues section of the repository and see which issues you're ready to tackle. Pick one, start your legendary battle, and once you've conquered the bug, submit a pull request.
