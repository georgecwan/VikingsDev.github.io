In this workshop, we will be creating an idle/clicker game using HTML, CSS, and JavaScript. Experience with said languages is recommended but not required. The project presented is meant to allow for creative expansion off of the framework provided, so don't be afraid to experiment or add on your own unique touch.

Note: Due to time interests, functionality and syntax will not be explained in full detail.

This is what we will be creating in this tutorial. The full source code for this demo will be available at the end of the workshop.


Project Setup


Before we do anything else, we must prepare the assets and set up a document structure. I would highly recommended downloading and setting up a proper IDE or code editor such as Visual Studio Code.

Assets and Files

For this workshop, we will be putting everything into a single file: index.html. In future tutorials, we will split up our projects into multiple discrete files.

The only assets we will be using in this workshop are images, which will be stored in the same folder as index.html for now. For this project we will need 3 images, all of which are available for download below. (Click on the image buttons to download)

JS Functionality

Insert the following text into <script> of index.html:

<script> in index.html
// Initializing the variables that we will need
let pencils = 0.0,
	pencilMakers = 0,
	pencilSharpeners = 1,
	pencilMakerPrice = 10,
	pencilSharpenerPrice = 10;

// The methods that are called whan the player presses the corresponding buttons

function makePencils() {
	pencils += pencilSharpeners;

	updateResources();
}

function buyPencilMaker() {
	// checking whether the player can buy a pencil maker
	if(pencils >= pencilMakerPrice) {
		// buying a pencil maker - note the order in which the statements are called
		pencils -= pencilMakerPrice;
		pencilMakerPrice += 3;
		pencilMakers++;

		updateResources();
	}
}

function buyPencilSharpener() {
	if(pencils >= pencilSharpenerPrice) {
		// buying a pencil sharpener - see corresponding comments above
		pencils -= pencilSharpenerPrice;
		pencilSharpenerPrice += (10 + pencilSharpeners);
		pencilSharpeners++;

		updateResources();
	}
}

// Adding the pencils created by the pencil makers
var tickPencils = setInterval(addPencils, 100);

function addPencils() {
	pencils += pencilMakers / 40;
	document.getElementById("pencil-count").innerHTML = "" + pencils.toFixed(0);
}

// Updating the document HTML to match the game code
// Note that this is not the most efficient way to sync the page

function updateResources() {
	document.getElementById("pencil-count").innerHTML = "" + pencils.toFixed(0);
	document.getElementById("maker-count").innerHTML = "" + pencilMakers;
	document.getElementById("sharpener-count").innerHTML = "" + pencilSharpeners;

	document.getElementById("pencils-per-click").innerHTML = "+" + pencilSharpeners;
	document.getElementById("pencils-per-second").innerHTML = "+" + (pencilMakers / 4).toFixed(2);

	document.getElementById("maker-cost").innerHTML = "" + pencilMakerPrice;
	document.getElementById("sharpener-cost").innerHTML = "" + pencilSharpenerPrice;
}

Congratulations! You have successfully created an incremental game in JavaScript. Be sure to take some time to analyze the code and experiment with it before moving on to the next tutorial!
